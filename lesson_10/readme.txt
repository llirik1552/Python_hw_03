Оба теста использует классы Page Object
1. Приложение "Калькулятор"
test_calculator.py использует класс CalculatorPage с одноименного файла
в файле прописаны методы:
Открытие браузера (значение - адрес url)
Установка задержки (в секундах)
Нажатие на кнопку (выполнение функции)
Получение результата (число)

Возвращаемые данные - число (сумма двух чисел)

2. Приложение "Интернет магазин одежды"
Базовая страница (открывает браузер, подгружает адрес, ждет указанное время)
Авторизация (имя и пароль)
Добавление выбранных вещей в корзину (выбор необходимых покупок и добавление их в корзину)
Заполнение данных для отправки заказа (сведения для оформления заказа)
